# CodeWithGPU

## Installation

### Install From Source

Clone this repository to local disk and install:

```bash
cd codewitgpu && python setup.py install
```

### Install From Git

You can also install it from remote repository: 

```bash
pip install git+ssh://git@github.com/seetacloud/codewithgpu.git
```
